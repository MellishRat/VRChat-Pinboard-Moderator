# VRChat Pinboard Moderator

This is the GitHub release package for the VRChat Pinboard Moderator.

Included files:

- `pinboard_moderator_gui.py`
- `pinboard_moderator_gui.spec`
- `flagged_words.json`
- `Run_Pinboard_Moderator.bat`
- `Build_EXE.bat`

`flagged_words.json` controls which words and URL fragments are highlighted locally inside the moderator app.

It does not block posts.
It does not delete anything automatically.
It only controls local highlighting.

A fuller README can be added to the GitHub repository.
